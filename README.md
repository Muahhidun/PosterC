# PosterC - Courier App

Мобильное приложение для курьеров Poster POS.

## Функциональность

- Авторизация курьеров по номеру телефона
- Просмотр доступных заказов
- Принятие заказов в работу
- Отслеживание времени доставки
- Завершение заказов
